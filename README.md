#FLG-QuestSystem

## Build

Run `dotnet build -tl` to build the solution.

## Run

To run the web application:

```bash
cd .\src\Api\
dotnet watch run
```

Navigate to https://localhost:5001. The application will automatically reload if you change any of the source files.
